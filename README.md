**DesignRoom** — учебный 3D-редактор комнаты на Unity.  
Пользователь может создать комнату, выбрать мебель из категорий, размещать объекты по клеткам пола, перекрашивать стены и поворачивать предметы.

Проект создан для практики работы с интерфейсом, объектами сцены, материалами, размещением по сетке и базовой логикой редактора.

---

## 🎮 Основные возможности

- 🧱 **Создание комнаты** — генерация комнаты по заданным размерам.
- 🪑 **Категории объектов** — выбор предметов из трёх категорий: мебель, рабочее место и декор.
- ⚙️ **Размещение по сетке** — установка объектов по клеткам пола.
- 🔄 **Поворот предметов** — поворот выбранного объекта на 90°.
- 🧽 **Покраска стен** — применение выбранного материала к стенам.
- ❌ **Удаление объектов** — удаление размещённых предметов по клику.

---

## 🕹 Как использовать

1. Введите **название проекта** от 3 до 8 символов.
2. Задайте размеры комнаты не меньше **3×3**.
3. Нажмите **Start**, чтобы создать сцену.
4. Используйте инструменты редактора:
   - **Material** — выбор материала для стен.
   - **Objects** — выбор категории мебели и размещение предметов.
   - **Rotate** — поворот объектов.
   - **Delete** — удаление предметов.
5. Клик по полу размещает выбранный объект по центру клетки.
6. Клик по стене применяет выбранный материал.
7. Кнопка **X** возвращает в главное меню.

---

## 🚀 Запуск проекта

1. Склонируйте репозиторий:

```bash
git clone https://github.com/qwertynoqwerty/DesignRoom3D.git
```

2. Откройте проект в **Unity 6.0+**.
3. Дождитесь импорта ассетов.
4. Откройте основную сцену проекта.
5. Нажмите **Play**.
